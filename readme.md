# Csharp Protobuf

## Simple Protobuf - LoginPacket

- Code
- Size
- SizeOfUsername
- Username
- SizeOfPassword
- Password

## How to compile protobuf

```sh
protoc --csharp_out=. protos/*
```

## How to install protobuf package from NuGet

```sh
dotnet add package Google.Protobuf --version 3.9.1
```
